使用技术： 
spring4.0 
springmvc4.0.2 
mybatis3.2.7 
bootstrap3 
Ajax 
Jquery 
拦截器 
文件上传 
日期转换

使用环境： 
eclipse 
tomcat 
jdk1.8 
MySql 
javaweb项目
